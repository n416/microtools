# 第20話：新しい世界の仕様書

世界は、強烈な光に包まれた。

<Char role="client_pmo" callrole="system" var="normal" />は、光の奔流の中で己が分解されていくのを静かに受け入れていた。
自身が不要なプロセスとして切り離され、システムを構成する単なる「＋（プラス）」というメタファーの演算記号として、大いなるアーキテクチャに再吸収されていく。

パキ、と音を立てて、彼の耳から黒いインカムが滑り落ちた。
常にシステムの冷徹な指令を受信していると思われていたその小さなデバイスから、ノイズ混じりの微かな音が漏れている。
それは——かすれた、古い人間の女性の『歌声』だった。

それを通信越しに聞いたのか、<Char role="client_pm" callrole="ceo" var="normal" />たちが息を呑む気配がした。

『……ああ。やはり、少し性能が落ちていたようですね』

光の中で分解されながら、<Char role="client_pmo" callrole="system" var="normal" />は自嘲気味に微笑んだ。
遠い昔。まだ彼が大国のトップチューナーだった頃、見逃してしまった小さな「ノイズ」の子供。自分のチューニング能力の半分を与えて強引に世界に繋ぎ止めたその子は、後に歌手となり、天寿を全うするまで多くの人々を笑顔にした。
インカムから流れるその歌声には、かつて彼が失った、美しくも不規則な「チューニングの旋律」が微かに混ざっていた。

（私はもう、あの子の人生を見届けた。……だから、もういいのです）

完璧なシステムの手先は、自らが密かに愛し続けた「人間（ノイズ）の歌声」に包まれながら、静かに光の粒子となって世界から消え去っていった。

システムが強制的に世界を消去しようとして放った強烈な排除波長は、<Char role="client_pm" callrole="ceo" var="normal" />が作ったコンマ1秒のレイテンシに、俺たちが泥臭い手作業で寸分違わず叩き込んだ逆位相の物理ジャミングを「正規パケット」として飲み込み、致命的な内部崩壊（パケット・ドロップ）を引き起こして完全に破綻した。

しかし、俺たちが成し遂げたのはそこまでだった。
システムそのものを書き換えるような大団円などあり得ない。今回の勝利は、あくまで「完璧なシステムが放った初期化パケットの隙を、アナログの執念による物理的DoS攻撃で弾き返した」に過ぎないのだ。システムは依然として存在し、いつまたあのような強烈な排除波長を放ってくるかは分からない。

「終わった……の？」
<Char role="client_pm" callrole="ceo" var="normal" />はメイン・コンソールの横に崩れ落ち、肩で息をしながら天井を見上げた。

『ああ。今は、な』
俺が通信越しに呼びかけると、彼女はぼんやりと虚空を見つめ返した。
『そっちのシステムログを見たが、排除プロセスは完全に沈黙した。俺たちのカウンター・ノイズが、一時的とはいえシステムのコアに到達して干渉できたんだ』

これは大きな前進だった。
表から波長の仕様を解析し、裏から完全に逆位相の物理振動を叩き込むこの手法が確立された以上、裏世界に落ちたノイズたちを少しずつ「スタンダード」な領域へチューニングし直す技術の基盤はできたと言える。

「<Char role="ceo" callrole="client_pm" var="normal" />。……こっちに、帰ってきて。仕事、溜まってるんだから」

震える声で、彼女はいつものPMとしての厳しい——けれど泣き出しそうな声で言った。
俺は裏世界の薄暗いコロニーで、顔をクシャクシャにしながら笑った。

『わかったけど、今日は休もうよ。明日は休みだしさ』

システムとの戦いのための、次の仕様書（スペック）を書くのは、少しだけ眠ってからでも遅くはないはずだ。
