# 第1話：システム障害の現状と課題

## 1. はじめに
本資料は、現在進行中のプロジェクトにおけるシステム構成と運用マニュアルである。
……というのは建前で、本当は最近起きている「妙なこと」を整理するための、自分用のメモとして残している。

## 2. 充実した、けれど少し奇妙な日常
今日もオフィスは活気に満ちている。
プロジェクトのスケジュールは適正に管理されており、決して世間で言うような「<Term id="death_march">デスマーチ</Term>」ではない。社長の<Char role="ceo" callrole="main_prog" var="normal_san" />は常に現場の意見を尊重してくれるし、先輩の<Char role="senior_eng" callrole="main_prog" var="normal_san" />のは的確で勉強になる。の<Char role="client_pm" callrole="main_prog" var="normal_san" />もの段階から積極的に協力してくれている。
<Char role="main_prog" callrole="main_prog" var="normal" />、<Char role="main_prog" callrole="main_prog" var="full_nospace" />も、メイン<Term id="engineer">プログラマ</Term>としてやりがいのあるを任せられ、チームの役に立てているという確かな実感があった。仕事は順調だ。

「す〜は〜...」

スマートウォッチから流れるメトロノームの音。
社長の<Char role="ceo" callrole="main_prog" var="normal_san" />が「<Term id="github">GitHub</Term>でたまたま見つけたんだけど、これ妙に集中できるんだよ」と勧めてくれたアプリの音だ。しかしなぜか最近、この一定のリズムに合わせて呼吸を整えないと、自分の意識が、あるいは世界そのものが薄れていくような奇妙な感覚に陥る。

体調は悪くない。むしろ頭も体も絶好調なはずだ。寝不足でも過労でもない。
毎朝買っているコーヒーも普通にうまい。
それなのに、オフィスの自動ドアが<Char role="main_prog" callrole="main_prog" var="normal" />にだけ反応しなかったり、すれ違いざまに人にぶつかってもまるで空気のようになんの感触もなかったりすることがある。

これは一体、何が起きているのだろうか。
