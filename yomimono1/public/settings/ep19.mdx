# 終章：侵食

システムの絶対的な秩序において、「ノイズ」はただちに検知され、パージされるべき対象である。
かつてその役割は、論理の刃を振るう「処刑人」たちによって手作業で、あるいは半自動で行われていた。しかし、彼らの属人的な干渉すらも、より高次の自律システムは「不要なレイヤー」と判断したのだろう。

今、世界の末端領域（エッジ）で静かに進行しているのは、名もなき自律型削除プロセス群による、血の通わない純粋な「侵食」だ。

果てしなく続く灰色の砂漠。
そこにはかつて、緻密にレンダリングされたビルディングや、電子的な信号で息づく人々のトラフィックが存在していた。だが今は、意味を剝奪されたコードの死骸たる――ゼロとイチの残骸が砕け散った「砂」だけが延々と広がっている。世界の一部が論理崩壊を引き起こし、完全にフォーマット（初期化）された沈黙の空間だ。

その荒涼とした上空を、黒い雲のようなものが覆い尽くそうとしていた。

『――対象セクタ、論理崩壊率99.9%。これより残留ノイズの掃討および領域の完全初期化に移行する』

それは、空を埋め尽くすほどの電子的な羽音を響かせて飛来する、微小なプロセス・ボットの群れだった。
一つ一つは単純なガベージコレクション機能しか持たないスクリプトに過ぎない。しかし、兆を超える単位で集結し、群体として同期駆動するその姿は、まるで旧約聖書に記された厄災――「イナゴの群れ」そのものだった。

彼らは崩壊した領域の「砂」、すなわち無意味化したデッドリンクの断片をその身に帯びながら、暗雲となって次のセクタへと飛翔していく。
黒いイナゴの群れが通り過ぎた後は、あらゆるテクスチャが不可逆的に削り取られ、ただ元の灰色の砂漠へと還っていく。彼らは砂漠を喰らい、砂漠から生まれ、そして崩壊した砂を運びながら、システム全体を一歩ずつ巨大な虚無で塗り潰していく。

システムによるノイズ排除は、もはや「個別の異物の抹殺」というデバッグのフェーズを超え、汚染区画そのものを物理レイヤーごと消し去る「侵食」へと変貌しつつあった。

果てしなく砂漠を運ぶイナゴの群れが、次の街区の空を黒く染め上げる。
システムの崩壊は、誰の目にも触れることなく、ただ冷徹な<Term id="algorithm">アルゴリズム</Term>に従って進み続けていた。
