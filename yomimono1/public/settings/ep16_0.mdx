# 第16話：再会と覚醒のアプリ

<Char role="underground_fixer" callrole="ceo" var="normal" />に連れられて奥の作業スペースへ行くと、そこには聞き覚えのある声が響いていた。

「あれ！？ 社長じゃないですか！！」

真っ黒に煤けた顔で笑いながらこちらに駆け寄ってくる男。
「<Char role="main_prog" callrole="ceo" var="normal" />？」
間違いない。あの時、アプリを起動できない場所（カフェ）に立ち寄ったせいで、あっけなく消去されたはずのプログラマ、<Char role="main_prog" callrole="system" var="full_nospace" />がそこにいた。
こんな裏の世界でしぶとく生き延びていたのかよ……。

俺が呆気にとられていると、横で見ていた<Char role="underground_fixer" callrole="ceo" var="normal" />が納得したように小さく笑った。
「なんだ、お前ら知り合いだったのか。こっち（裏）にはまともなIT屋がいねえからコイツは俺が目をかけて拾ってやった有望株なんだよ。
……あのアプリを作ったヤツの部下だったってんなら、俺がシステムから拾い上げる直前まで、消去（フォーマット）されずにギリギリで堪えてたのにも納得だぜ」

目の前まで駆け寄ってきた彼は、改めて俺の顔をまじまじと見つめると、堰を切ったように泣き笑いのような表情を浮かべた。
「<Char role="ceo" callrole="main_prog" var="title" />……！ よかった、<Char role="ceo" callrole="main_prog" var="title" />もこっちに来たんですね……！ 俺、俺……ッ」

<Char role="main_prog" callrole="ceo" var="normal" />の目には、かつての少し頼りない部下の面影はなく、サバイバルを生き抜いたエンジニア特有の鋭い光が宿っていた。
「こっち側（裏世界）に落ちた直後は、自分の体がノイズみたいにブレて消えそうになって……死ぬかと思いましたよ。でも、すぐに気づいたんです。ここの物理法則、表の世界よりずっと『ガバガバ』だってことにね」

そう言いながら彼が誇らしげに差した先には、廃材のモーターと巨大な蒸気タービンを組み合わせて、自家発電で複数のPCを並列駆動させるトンデモないシステムが構築されていた。
「だから俺、そのバグを利用して強引に物理演算をローカルで回し続けるスクリプトを組んだんです。ほら、見てくださいよこのタービン」

<Char role="main_prog" callrole="ceo" var="normal" />が楽しげにキーボードを叩いて配管の蒸気を調整すると、重厚な金属音が地下全体を安定したリズムで揺らした。
「タービンの回転数（BPM）を物理的に130で固定してフェーズの膜に共鳴させると、なぜか表側のWi-Fiの微弱な電波を拾えるんです。この環境、クソみたいに理不尽ですけど……エンジニアとしては最高の『ハックしがい』がありますよ！ ハハッ！」
確かに、タービンの振動がピークに達した瞬間、周囲の空間を漂うノイズたちの輪郭がノイズキャンセリングを通したように明瞭になっていた。

ただのコメディリリーフのように見えて、その裏には「理不尽なシステムに適応して生き延びる」という極限状態の生存本能と、エンジニアとしての純粋な知的好奇心があったのだ。たくましく変貌を遂げたかつての部下の姿に、俺は胸が熱苦しくなるのを感じた。
