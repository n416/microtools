# 第01話：木屑の匂いと、甘い期待

　父親の工房は、いつも甘い木屑の匂いがした。
　午後からの西日が、木枠の窓を抜けて宙を舞う細かい塵を金色にきらきらと輝かせている。アルトは竹箒を握り、床に散らばった鉋屑（かんなくず）を器用に、そして素早く部屋の隅へと集めていく。くるくると丸まった薄い木片が、箒の先でカサカサと小気味良い音を立てた。

「アルト、相変わらず手際がいいな。箒の使い方がもう職人みたいだぞ」
「このくらいの事、なんてことないよ。足の裏に木屑が刺さるって、母さんが怒ってたからね」

　作業台に向かっていた父親が、手を止めて目を細めた。彼の太い指にはいくつか古傷があり、着古したエプロンには木工用の油が染み付いている。しかし、アルトに向けられるその眼差しは、いつだって温厚で優しかった。アルトの手先の器用さと身軽さは、木工職人である父から見ても文句のつけようがないものだった。

「ほら、こっちは仕上げが終わったぞ。どうだ？」

　父親が作業台から持ち上げたのは、滑らかに磨き上げられた木彫りの小鳥の置物だった。

「すごい！ これ、ニーナへのプレゼント？」
「ああ。仕上げに『防腐の魔法陣』の線を薄く彫り込もうとしてるんだが、どうも本に載ってる平面図の通りにいかなくてな」

　父親が見せてきた木彫りの底には、複雑な幾何学模様が下書きされていた。
　アルトはその平面の線画を一見しただけで、頭の中に『動く立体の構造』がパッと組み上がるのを感じた。アルトにとって魔法の陣は、大人が暗記するような平べったい記号や呪文ではない。どこから力が激しく流れ込み、どううねりを上げて渦を巻き、何秒後にどの線と交差するのか——まるで一流のアスリートが空中での姿勢制御を本能的に把握するように、その『立体的な形と、時間軸に沿った動き』が手に取るように理解できたのだ。

「父さん、そこ違うよ。本だと平面的に描かれてるけど、ここの線は少しだけ浮かせて、三秒後に奥の円と螺旋状に絡み合うように意識して彫らないと、力が逃げちゃうんだ」
「なっ、螺旋状に絡み合う……？ お前、まだ学校で魔法陣の基礎すら教わってないはずだぞ。なんでそんな立体や時間ごとの動きが分かるんだ？」
「なんでって……普通に、そう動くのが一番しっくりくる形に見えるからだけど」

　アルトが不思議そうに首を傾げると、父親の大きな手がアルトのザンバラな黒い髪をわしゃわしゃと、少しの驚きと極大の期待を込めて撫で回した。

「……お前も、明日でいよいよ十歳だな」

　父親が少しだけ誇らしげに息をついた。
　十歳。それは子供が「魔力経路」を安定させ、学校で公的にちょっとした魔法の練習が解禁される特別な年齢だ。

「父さんも、僕くらいの頃は魔法の授業が楽しみだった？」
「そりゃあな。父さんはごく普通の凡人だったから、お前みたいに直感で『魔法の立体構造と時間の流れ』が見えるような才能はなかったがな。お前のそのアスリートみたいな運動神経や直感を見ていると……案外、歴史に名を残すような大魔導士になっちまうスゴいやつなんじゃないかって、父さんちょっと期待してるんだぞ」

　そんな他愛のない会話をしていると、不意に工房の裏口が勢いよく開いた。

「お父さん！ お兄ちゃん！ もうご飯よ！ いつまで木を削ってるの！」

　夕暮れの風と一緒に飛び込んできたのは、九歳になる年子の妹、ニーナだった。少しだけ背伸びをしたようなおませな口調で腰に手を当てて怒ってみせるが、アルトの顔を見た途端にパッと顔を輝かせた。

「お兄ちゃん、今日の夜ご飯、ハンバーグだって！ アルトの十歳のお祝いの前夜祭だから特別よ！」
「ほんと！？ やった！」

　アルトも無邪気に声を上げ、ニーナがアルトの腕にぎゅっと抱きついてくる。その無邪気な笑顔と、兄に対する純粋な親愛の情を見ているだけで、アルトの胸の奥もじんわりと温かくなった。

「行こうぜ、ニーナ。特製ハンバーグが冷めちゃうぞ」
「うんっ！」

　夕焼けが家族の影を長く伸ばしていた。
　いつもの食卓。不恰好に丸められたハンバーグと、湯気を立てる野菜のスープ。ニーナが学校の出来事を身振り手振りで話し、父親が仕事の失敗談を大げさに話しては全員を笑わせる。
　母親がふわりと笑ってアルトにお茶を注いでくれた。

「明日の学校での『測定』、楽しみね。アルトは何でも上手にできちゃうから、魔法の才能もきっとすごいに決まってるわ」
「ええ〜、そんなことないよ。普通が一番だよ」

　照れ隠しで笑いながらも、アルトの心臓は期待に小さく跳ねていた。
　明日になれば十歳になる。自分も魔法が使えるようになって、何かあったらこの優しい両親を助けたり、ニーナに魔法で面白い光を見せて笑わせたりしちゃうんだ。
　アルトはその賑やかで温かい食卓で美味しいハンバーグを頬張りながら、ささやかで絶対的な幸福に包まれた『明日』が来ることを微塵も疑っていなかった。
