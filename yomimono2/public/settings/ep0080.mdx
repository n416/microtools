# 第08話：泥水の天井と、味のしない流動食

　地下スラムの最下層。腐敗した土と錆水の下水が混じり合う異臭の中、今にも崩れそうな泥壁のあばら家で、ミアは退屈そうに古びた木箱の上に腰掛けていた。

　死に絶えた灰色の瞳の先には、カビの生えた薄い毛布にくるまり、三日三晩ほど泥のように眠り続けている男の姿がある。
　これが、つい数日前まで王立学園の天才特待生だったアルト・エヴァンスの成れの果てだと言われても、到底誰も信じないだろう。

（……ようやく、魔力経路の焼き焦げが落ち着いたか）
　ミアは彼の微弱な鼓動を計測しながら、ひどく冷め切った頭で思考していた。
　魔導炉事故での、あの異常な二十五年の寿命の前借り。人間であれば確実に即死するほどの過剰な魔力出力を、この男は『極太の魔力経路』というただ一点の異常な才能だけで耐え抜いた。
　彼女が三百年の人生で出会った、最高峰の理論才能にして最狂の論理的欠陥品（異端）。

「……あんたが、僕を助けてくれたのか」

　しゃがれた枯れ声に、ミアが視線を下ろす。
　毛布の中でようやく意識を取り戻したらしいアルトが、自分のひどく老婆のように皺張った手をぼんやりと見つめながら、暗い天井を仰いでいた。

「勘違いするな。死に損ないの欠陥品を観察対象として拾ってやっただけだ」
　ミアは一切の抑揚のない、氷のように冷え切った声で吐き捨てた。
「妹の治療の際、お前は無意識下で『私の三百年分の術式構築』を瞬時に論破し、組み替えてみせた。……その魔法構造の極上の天才が、なぜ自ら進んで命をドブに捨てるのか。お前のその理解し難い『絶対的な矛盾』を解明し尽くす。私はそのためだけに、お前の残りの寿命に張り付くことにした。光栄に思え」
「……そっか。観察対象か」

　だが、突きつけられた黒い傘の切っ先を前にしても、アルトの疲弊しきった顔には一片の感情も浮かばなかった。
　彼は焦点の合わない目で泥水の染みた天井を見つめたまま、ゆっくりと毛布を被り直す。
「ありがとう。ちょうどよかったよ」
「何がだ？」
「全てを失って、バケモノ扱いされて、もう帰る場所もない。今の僕には……これくらい誰からも同情されない、冷たい場所の方が、お似合いだからさ」

　言葉の裏にあるのは、極限の逃避と、完全なる絶望だった。
　王立院の幹部候補として背負っていた重圧も、両親からの温かな無償の愛も、彼が全てをベットして守り抜こうとしたものは、自分の手によって粉々に砕け散った。
　燃え尽きた灰のような無気力。アルトはそのまま壁の方へと寝返りを打ち、二度と起きあがろうとはしなかった。

「……死にたければ勝手にしろ。だが、私の観察対象である以上、飢え死になどという非効率な最期は許さん」

　ミアは忌々しげに立ち上がると、スラムの怪しげな市場から調達してきた泥のような謎の素材を魔法の火で一瞬で分解し、完璧な栄養素だけを抽出した『ひどい匂いのするドス黒い流動食』を生成した。

「おい、口を開けろバカ」
「……いらない」
「開けろ」

　それは、三百年の魔女と、寿命を燃やし尽くした初老の少年の、奇妙で冷え切った同棲生活の始まりだった。
　アルトはエリートとしての全ての光を失い、薄暗いあばら家の底で、ただ無気力に味のしない流動食を飲み込むだけの、完全なる『引きこもり（生ける屍）』へと成り下がったのだった。
