# 0180『過去への術式と、帰る場所』

　スラムの地下空間は、今や別次元のような凄まじいマナの奔流に包まれていた。
　ミアが掲げた手の中で、ヴィランの死の果てに結実した黄金の種子——『魔力還元炉の生体コア』が激しく脈打っている。
　そして、ミア自身の体からは、彼女が三百年間『自分の命は他人のためには絶対に一秒たりとも燃やさない』と誓い、手付かずのまま温存し続けてきた【彼女自身の莫大な本来の寿命＝百年紀】が、凄まじい勢いで放出され続けていた。

「パモナナッ！！ 繋がっているんだろう！ 見ろ！！」

　ミアは空中に巨大な『睡蓮』の通信回路を無理やりこじ開け、遥か遠くの森の国へと叫んだ。
　空間の歪みの向こうで、深く皺の刻まれた老婆——パモナナが、驚愕と慈しみの入り混じった顔で通信窓を覗き込んでいる。

『……ええ。見えますよ、ミア。……間違いありません。その黄金の種は、間違いなく私の師匠（ジーナ）が夢見た、完成された還元炉の心臓。……そしてそこへ倒れているアルトの経路は、今も完全に適合（アクセス）を待っている』
「ならばよし！ 私の【百年分】をブースト燃料として全て叩き込み、このコアを【過去】のアルトへ撃ち込む！」

　パモナナは思わず息を呑んだ。
　それは、世界で最も天才的で、そして最も『命価効率の悪い（非合理的な）』ただの力技の禁呪だった。

　ミアが黄金のコアを空に高く掲げた。
　その瞬間、スラムの地下の空気が「パキンッ」と甲高い音を立てて割れた。
　空間の奥底から吹き込んできたのは、地下にはあるはずのない『夕暮れの茜雲の匂い』と、遠くで犬がキャンキャンと元気よく吠える声だった。

『……あの日へ。彼が初めて小さな命を救おうと決めた、あの原点の時間に……炉心を直結させたのですね』

「そうだ！ 過去から現在まで、コイツが誰かを治して理不尽に捨ててきた莫大な余剰生命力……そしてさっき、結実のために四十年分も不当に奪われた分！ その全てをこの『睡蓮（還元炉）』の逆流システムで、現在のテメェの肉体に叩き込む！」

　ミアは空に向かって、世界に対する絶対的な勝利宣言を響かせた。

「世界（システム）がこの大馬鹿から巻き上げた、数十年分の過払い金だ。……全額、利子付きで返還請求してやる！！」

　ミアが自身の胸を強く叩くと、彼女の十六歳の肉体から、炎のような極彩色のマナがごっそりと引き剥がされた。
　――ズゥゥンッ！
　黄金の光が空間の裂け目から逆流し、瓦礫に倒れ伏す九十代の姿のアルトへと突き刺さる。光は彼を焼くのではなく、深いシワを撫でて消し去り、枯れ木のような手足に熱を吹き込んでいく。
　死に向かっていた時計の針が、物理的な逆転現象として視覚化されていた。だが、奇跡の代償は重かった。

「待って……」

　瓦礫に倒れ伏していたアルトが、光に包まれながら血を吐くような声で彼女のローブの裾を掴んだ。
　かすれきったその瞳には、彼自身が回復していくことへの喜びではなく、明確な拒絶の光が宿っていた。

「ミア……！ お前、髪が……っ！」
「……」

　アルトが若返っていくのと反比例するように、光を送り出し続けるミアの美しい銀髪が、根元から真っ白な老婆のような白髪へと急速に色褪せていく。透き通っていた肌に、年相応の人間らしい疲労と血の色が混じり始めていた。
　彼女は天才だ。こんな男の言葉など無視して、コンマ一秒の計算で全てを終わらせることなど容易い。

　だが、急速に「ただの人間」へと落ちていく中で、彼女は発動を一瞬止めた。
　同時に世界の時間も止まったように感じられた。時空を操作する魔法が発動する時はこうなる筈という二人で編み出した理論の通りだった。

「それでも！お前が嫌なら、やめる」

　ミアは、泣き出しそうな顔で、けれど真っ直ぐにアルトを見据えた。

「私は、瀕死のお前の意思なんか無視して、勝手に計算を強行できる。……でも、やらない。そういう自分勝手な理不尽を嫌って、ずっと苦しんできた奴だから、お前は。私は！！ずっと自分勝手にやってきたんだ。それが嫌だった！」
「……」

　彼もまた、瓦礫の中で息を呑んでいた。
　三百年間、命の数字しか見てこなかった自分が、今どれほど理不尽で無様な泣き顔で『どうか死なないでくれ』と懇願しているか。
　彼にもきっと、痛いほど伝わっているはずだった。しかも、彼女自身を今まで守ってきた『百年の防壁』を全て投げ打ってまで。

　アルトは血を吐きながら、それでもはっきりとした声を出した。

「そんな莫大な生命力を、俺なんかに使っていいのかよ……！　お前は本当は、その命を使って……十六歳の、あの時代に戻りたかったんじゃないのか？　俺にそれを使ったら、お前はもう……あの時代には戻れなくなるかもしれないけど！？」

　それはミア自身がかつて語った、彼女の唯一の希望だったはずだ。過去へ戻って仲間と死ぬためだけに、彼女は三百年を生き汚く繋いできた。
　しかし、ミアは泣き顔のまま、ひどく柔らかな声で答えた。

「いいよ――構わない。私の魂のアンカーは、もう……今、ここにあるから」

　それは、過去の絶望に囚われていた三百年の魔女が、初めて『今の時代（アルトの隣）』を選んだ瞬間だった。
　その沈黙の中で、ミアは確かに感じ取った。アルトの中でずっと死守し続けてきた『自己犠牲の肯定（自分が損をすればいいという逃げ）』が、崩れ落ちていくのを。

「……なあ、ミア。俺、さ」
「何だ」

　極限の老いにまみれた顔で、アルトは震える手を伸ばし、ミアの冷たい頬に触れた。
　それは、彼が生まれて初めて口にする、本物の身勝手な『弱さ』だった。

「やっぱ……家に、戻りたかった」

　優しい両親と元気な妹がいる、温かくて、魔法なんて使わずに済んだ、あの普通の家族の待つ家に。
　それはミアがずっと見抜いていた、『他人のために綺麗に死んでいく満足げな顔』の奥底に隠し持っていた、ただの少年のたったひとつの無自覚な未練だった。

　ポタリ、と。
　ミアの目から、三百年間流したことのなかった大きな雫が、アルトのひび割れた頬に落ちた。

「あのさ、ミア……」
「……なん、だ」
「……お前が俺のために『百年』も払うなら……俺はそれを受け取って、ずっと生きる義務がある。……ここで綺麗に逃げるのは、なしだ」

　それは、アルトがミアから提示された【途方もない負債（呪い）】を、自らの意思で受け入れた瞬間だった。
　ミアは泣き顔のまま悪戯っぽく笑い、アルトと指切りをするように、その腕をきつく掴み返した。
