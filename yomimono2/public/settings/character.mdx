# キャラクター設定

## アルト（灰の少年）
- **実年齢**：16歳 / **外見**：40代
- **才能**：命価効率は人類最悪。しかしスタミナ（経路）は異常な極太で、数十年の命を一気に流しても死なない特異体質。この極太経路は魔力還元炉を受け入れられる世界唯一の適合器。だが、その経路の広大さゆえに、自己回路を調査して繋ぎ合わせる「契約魔法の同調作業」が通常の何十倍もの規模になり、いつ終わるとも知れない途方もない苦痛を強いられている。
- **成長の軌跡**：
  幸福な家庭で育ち、10歳の魔法解禁時に命価効率最悪の才能が判明。親に禁止されながらも目の前の痛みを見過ごせず使い続けた。妹の事故後はスラムに落ち、善意を武器として使われる構造を理解しながらも怒りと葛藤を抱えつつ正しく生きようとし続けた。
- **内面の核**：「うまく立ち回れば良いと頭では分かる。でも目の前で子供が泣いていたら計算なんてできない」
- **秘めた願望**：誰にも言えない「家には戻りたかった」という気持ちをずっと抱えている。
- **記憶の二重性**：転換炉による過去の最適化後、改変された幸せな記憶と泥水をすすった記憶が共存する。その両方が自分だと統合することが真の成長になる。

## ミア（三百年の魔女）
- **実年齢**：300歳 / **外見**：16歳
- **才能**：命価効率は0.01倍の超限界効率。理論上、数千年以上を生きられる神懸かり的な燃費を持つ。
- **装甲（寿命の内訳）**：自分自身の莫大で無傷な残り寿命（本来の寿命約60年分＝実質数千年分）。魔法を使う時は契約（ヴィランの搾取システムと同種）を用いてクライアントに全コストを払わせ、自分の命はコンマ一秒も使わずに温存し続けてきた。
- **行動原理**：徹底した自己防衛と効率主義。愛する仲間が死んだ絶望から心を守るため、他人の命の数字（燃費）だけを管理する冷徹な詐欺師（魔女）として生きてきた。
- **契約魔法の習熟**：高度な契約魔法を使いこなすが、本来は回路の同調に伴う激痛に耐えられる性質ではなかった。しかし、彼女の持つ異常な「絶望から這い上がるための狂気じみた根性」によって、数年がかりで血を吐くような痛みに耐え続け、自己の回路の掌握を気合で終わらせたという泥臭い過去を持つ。
- **変化の軌跡**：
  アルトへの軽蔑→苛立ち→防壁の決壊という三段階を経て、300年間封じてきた感情を取り戻す。最後に計算をやめることが、効率という「正当化の論理」を捨てる瞬間として機能する。
- **エピローグ**：残り寿命が「人並みになった」とだけ感じている。正確な数字は知らない。計算するのをやめた。

## ヴィラン（悪徳仲介屋）
- **才能**：アルトと同じ極太のスタミナを持つ男。さらに「生まれつき痛みを感じづらい」という特異体質。
- **役割と行動原理**：かつて妻を救うために無茶をして道を外れ、搾取システムに乗った瞬間に怪物になった「アルトの最悪のIFルート」。当初の目的は「妻の救済」であったが、他者を犠牲にしてまで生き長らえさせようとした結果、当の妻から「化け物」と激しく拒絶されて死別した。そのため現在の真の目的は妻の蘇生などではなく、「極限状態になれば誰だって他者を犠牲にする。あの時の俺の選択は正しかったのだ」と世界に証明し、過去の自分を自己正当化することへと完全に『すり替わって』いる。
- **アルトへの強烈な同族嫌悪**：他者のために自分をすり減らして平然と笑っているアルトの「無償の善意」を肯定してしまうと、過去に他者を犠牲にした自分が完全に悪だと認める（過去の選択の否定）ことになってしまう。そのためアルトの存在を絶対に許さず、屈服させて他人の命を奪わせようとする。二人の物理的な対立は、存在意義を賭けた「生き方の代理戦争」となっている。
- **退場**：アルトからの「お前の奥さんは、本当に幸せそうに笑っていたのか」という痛烈な問いによって自己正当化の城郭が崩壊。直後、ミアの幻影術式で過去の病室を見せられ、心の底に封印していた「あの時、他人の命を奪うのではなく、自分の命を失ってでも妻を助ければ良かった」という本当の『後悔』を呼び覚まされる。幻影の中で、今度こそ他人を犠牲にせず自らの命だけを捧げる決意と共に、悔恨と安堵の入り混じった顔で睡蓮へ全生命力を注ぎ込み、自死を遂げる。

## パモナナ（老婆の魔法使い）
- **実年齢**：300年以上 / **外見**：老婆
- **素性**：ミアと同世代の長命者。旧知だが深い友情ではなく、長く生きすぎた者同士の乾いた同士感覚。
- **老いの理由**：命を延ばす研究のために自分の寿命を300年かけて少しずつ実験台にしてきた。魔力還元炉の設計者。
- **怒りの深さ**：良い人間ほど早く消費され死んでいくことを古い時代から見抜いていた。弟子たちも師匠も自己犠牲をしてどんどん死んでいった。残ったのは人の命を啜る者ばかり。その怒りは慈しみに還元された。
- **ジーナへの想い**：「先生は最後まで、自分の睡蓮がどう使われているか知らなかったと思いますよ。……それで良かったと、私は今でも思っています」還元炉はジーナの夢の、正しい形での完成。名前の由来を知る人間は全員死んでいる。

## ジーナ（パモナナの師匠・故人）
- **役割**：命喰みの睡蓮の真の作者。
- **思想**：純粋だった。命価効率の格差という理不尽を見て怒り、その怒りを研究に変えた。魔力を平均化できれば才能のない人間が50倍の寿命を失わなくて済む。それだけを考えて睡蓮を作った。
- **結末**：悪用されるとは思っていなかった。人間をそこまで疑っていなかった。自分の睡蓮がどう使われているか知らないまま死んだ。善意から生まれ、善意のまま死に、しかし発明だけが残って搾取され続けた人間。
