# 第06話：数百年ぶりの重低音

<!-- POV: ミア -->
　街の反対側にある、打ち捨てられた時計塔の薄暗い陰。
　紺色のローブを深く被ったまま、ミアは退屈しきった視線で王都の大通りを行き交う人々を見下ろしていた。
　彼らの目には、ミアがただの『十六歳の小娘』に見えているはずだ——中身が氷の魔女だとも知らずに。

　彼女の目には、人々の笑顔も、怒りも、悲しみも、すべては無味乾燥な『命価効率の数字』の羅列としてしか映らない。
　あそこで笑い合いながら歩く恋人たちは、無駄な心拍数の上昇と体温の発散により三日分の寿命を前借りして消費している。路地裏で怒鳴り合う商人たちは、過度な感情負担という劇毒で己の魔力経路を錆びつかせ、一ヶ月分の時間を自らドブに捨てている。
　馬鹿で哀れな生き物たちだ、とミアは冷ややかに鼻を鳴らす。
　彼らは誰も、彼ら自身の命がどれほどの速度で燃え尽きようとしているのか、その正確な『残量の砂時計』を計測しようともしないのだ。

　自らの残された『三世紀に及ぶ底なしの寿命』を、ただの一秒もすり減らしたくはない。そのためならば、息を潜めるように退屈で枯れ果てた時間をやり過ごすことも彼女にとっては苦ではなかった。
　呼吸の回数を制限し、歩く速度を常に一定に保ち、感情の起伏すらも脳内の何重もの術式で抑え込む。誰かを愛することも想うこともなく、ただ観測者として息を潜め続ける。
　一日に許容される魔力消費の最大値は〇・〇一パーセント。それ以上の出力は、彼女にとって死の浪費と同義だった。
　何百年もの間、安全圏から世界を傍観し続けるための氷の防壁を、ミアは己の内側に築き上げている。

　ただただ代わり映えのしない暗く長いトンネルの中を、一定のリズムで歩き続けているような、色のない日々。
　明日も、明後日も、百年後も。この退屈で『計算通り』の感情の抜け落ちた時間が永遠に続くはずだった。——少なくとも、一秒前まではそう信じきっていた。

　——だが、その時だった。

　ズゥゥンッ……と。
　空間の底（位相）そのものを激しく真っ向から揺さぶるような、恐ろしく分厚い熱波の波動が、ミアの精緻な計算式を真正面から粉砕した。

「……何、今の」

　ミアは灰色の瞳を大きく見開き、赤い閃光の残像が上がった遥か遠くの路地裏の方角を、反射的に鋭く睨みつけた。
　常人には見えず、大通りの人々は誰も気づかずに馬鹿みたいに笑い合って通り過ぎる。
　だが研ぎ澄まされたミアの感覚には、規格外の命力が『たった一瞬に凝縮された重低音』のような破裂として内臓を震わせたのだ。
　彼女の脳裏で途方もない速度の演算が行われる。弾き出された解析結果の数値に、ミアの思考は文字通り凍りついた。

　——ありえない魔法の圧縮率だ。
　本来、優秀な術者が額に汗しながらじんわりと燃やすべき人間数十日分の『命』を。
　その未知の術者は、たった一秒にも満たない瞬間に針の穴を通すように圧縮し、一挙に叩き直して爆発させたのだ。
　それはかつてミアが目にしたことがある、数百人の奴隷の命を炉心にくべ撃ち放たれた旧時代の巨大魔法兵器の砲圧にも似た、常軌を逸した密度だった。

　さらにミアの背筋を凍らせたのは、その『最低の効率』をもたらしている魔力経路そのものの異常性だ。
（……太すぎる。並の人間であれば内側から弾け飛ぶ熱量を、何の防壁も通さずに生身の導管から直接吐き出している……！？）

　ミアの脳内の計算盤が、その代償の重さを容赦なく機械的に弾き出す。
　たった今、あの未知の術者は、どこかの誰かのちょっとした命を治癒するためだけに行使したその反動で、自らの『百日』近い未来の寿命を一瞬にしてドブに投げ捨てたのだ。

（……馬鹿だ。狂っている。あんな無意味で自己犠牲にしかならない命の空吹かしを……一体何のために……っ）

　理解不能。論理破綻。最大の無駄。
　ミアの防壁が、激しい嫌悪感と生理的な拒絶反応で総毛立つ。あんなものに関われば、自分が百年かけて構築した精緻で美しい計算式など、あの熱波で一瞬にしてドロドロに溶かされてしまう。
　今すぐここから立ち去るべきだ。脳内の警鐘が理性として即座にそう命じていた。

　……なのに。

「あ……」
　ひどく乾燥してひび割れていた彼女の静止した時間に落とされた、たった一滴の熱い血。
　自分が百日の寿命を取り戻すために、どれほどの一切の熱を遮断する防壁を敷いてきたか。それをあの馬鹿は、どこかの小さな命ひとつのために、鼻歌でも歌うように『与えて』しまったのだ。
　無意識下で彼女の魂そのものがドクンと低く共鳴し、かつてない甘美な痺れとなって、足先から脳髄へと這い上がってくる。

　ミアは遥か彼方の暗がりを、ただ呆然と見つめ続けていた。
　終わりなき退屈の中で息を潜めていた彼女の鼻腔を、ふと掠めた純粋な命が焼け焦げるような匂い。
　（なんだ、これは……）
　ずっと冷え切っていたはずの胃袋の底で、数百年ぶりとなる激しい『飢え』の熱が、チリチリと灯り始めているのをミアは確かに感じていた。
