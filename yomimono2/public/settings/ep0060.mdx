# 第06話：二十五年の対価と化け物の烙印

　土砂降りの冷たい雨が、果ての見えない暗がりの空から降り頻る中。
　アルトが制服を泥まみれにしながら、弾かれたように王都の中央診療所へと駆け込んだ時、すし詰めになった廊下には重苦しい死臭と、鉄錆のように生臭い血の匂いが充満していた。
　怒号の飛び交う命の選別の混乱を潜り抜け、最も奥まった薄暗い治療室に辿り着いた先には、絶望の底に沈む両親の姿があった。

「助からないって……そんな……！ 嘘だと言ってください！ 何とかなりませんか……ニーナは、まだ十四歳なんですよ……！」
　粗末な床で、母親が髪を振り乱して泣き崩れていた。その横で、いつもは厳格な父親が、青ざめた顔で医師の白衣にすがりついている。

「ニーナ……？」
　黒タグを貼られ、血まみれでベッドに横たわる妹を一目見た瞬間、アルトの呼吸が止まった。
　華奢な胴体はあり得ない方向にひしゃげ、青白い皮膚の下で内臓が広範囲にわたって圧迫・破裂している。崩落した魔導炉から強烈な魔法震と瓦礫の直撃を真正面から浴び、彼女の生命活動は急速に失われつつあった。

「……通常の医術ではもう手遅れです。ご両親、覚悟を決めてください」
　年老いた重鎮の魔法医の言葉は重く、そして酷薄なほど絶望的だった。
「私が半年分の寿命を賭ければ、あるいは……だが、この未曾有の災害で、私の命と立場を一人の少女のためだけに捨てるわけにはいかん……彼女の心臓は、あと数分ともたない」
　それは、過酷な命の選別による絶対的な死の宣告だった。

「——僕がやる」

　アルトは震える声で呟き、自らの腕輪に手をかけた。
「アルト！？ ば、馬鹿なことを言うな！ お前の才能じゃあ、擦り傷一つ治すのにもどれだけ苦労するか……」
　父親が涙まみれの顔で止めようとした。だがアルトはその手を力強く振り払った。
　制服のポケットから、四年間隠し持っていた自作の合鍵を取り出す。そして、左腕にはめられた重厚な『魔力止めの腕輪』の鍵穴に、それを躊躇いなく突き刺した。

　魔力の経路が冷え冷えと結合する鈍い音が鳴る。
　物理的な封印が外れた瞬間、極太の魔力経路に、これまで堰き止められていた規格外のスタミナが、荒れ狂う激流となって爆発的に流れ込んだ。
　ゆっくりとスローモーションのように感じる静寂の中で、アルトの脳髄はベッドの上のニーナを、４次元の立体構造として恐ろしいほど鮮明に把握し始めた。

（だめだ……酷すぎる。壊死した肺の完全再生、千切れた大血管の無数な接合、粉砕された肋骨の再構築、魔力毒の完全浄化……ダメだ、規模が大きすぎる）
　アルトの演算能力が、冷酷なまでに正確な『代償』を弾き出していく。
　人類最悪の命価効率を持つ彼が、一瞬で彼女を完全に復元するために突きつけられた命の代償（対価）は——

（——『二十五年』）

　その解答が出た瞬間、アルトは絶望に言葉を失った。
　いま、まだ十六年しか生きていないのだ。これから生きるはずだった黄金のような青春も、特進生としての栄光も、すべてがいまこの瞬間に燃え尽きる。
（僕の人生が、終わる）

（————けど、それがどうした）
　頭の奥底で特進生の『計算』が叫んでいる。絶対に損だ、お前が死ぬぞ、と。
　だが彼の純粋すぎる心が、その特進生の計算式を容赦なく叩き潰した。

（二十五年の寿命で、ニーナの残りの何十年という人生が元通りになるなら────安すぎる！！）

　アルトは、自身の膨大な命価（寿命）を一気に魔力へと変換し、ベッドの上の妹へと直接叩き込んだ。
「やめろおぉぉっ！！ アルトォォ！！」

　途方もない質量の、眩いほどの純白の光が診療所を包み込む。それは痛々しいほどに美しい奇跡だった。
　ニーナの致死的な傷跡が瞬く間に塞がり、不自然にひしゃげていた手足が元の柔らかな形を取り戻していく。
　だがその裏側で、アルトの肉体を『理不尽な代償』が猛烈な勢いで侵食し始めていた。
　十六歳の細胞が、わずか数秒の間に二十五年分の時を強引に走り抜けていく。
　若々しい黒髪からは根本から色素が抜け落ち、あっという間に白髪交じりの灰色へと変貌した。
　——痛みは、何一つなかった。
　それこそが、命価消費というシステムの最も恐ろしい点だった。熱も、苦痛もなく、ただ自分の内側の「最も確かな重み」だけが、音もなくゴッソリと抉り取られていく絶対的な喪失感。
　肺から息を吐くたびに、生きる意味や、これから経験するはずだった喜怒哀楽のすべてが、虚無の風となってスースーと通り抜けていくような、圧倒的な空虚。
　関節の軟骨がすり減り、筋肉がたるみ落ち、十六歳の瑞々しい肌は水分を完全に失って乾燥し、深く無数の皺が刻み込まれても、肉体は悲鳴一つ上げない。一秒が重なり合うごとに、彼の中にあった『未来』という無数の可能性の枝葉が、ただ静かに、初めから存在しなかったかのように消え失せていく。

「……あ……っ、あ…………」
　あまりにも底知れない己の存在の『欠落』に、アルトの口からひゅうひゅうと虚無に震える空気の音だけが漏れ出す。
　光が収まる。極限まで削られ、たった三十秒あまりで約四半世紀の時間を失ったアルトは、空っぽになった抜け殻のように、両膝から呆気なく崩れ落ちた。
　ベッドの上では、信じられないことにニーナの血痕すらも完全に消え去り、安らかで健康的な寝息が聞こえ始めていた。

「……よかった……母さん、父さん。ニーナは、もう平気だよ……」
　アルトは軋む体を起こし、安堵の笑みを浮かべて両親を振り返った。
　喜んで抱きしめてくれると思っていた。
　だが振り返った先、両親の顔に浮かんでいたのは、歓喜ではなく——名状しがたい絶望と恐怖だった。

「……あ、あ、ああ……っ」
　母親は口を両手で強く押さえ、ガタガタと震えながら後ずさりし、壁にぶつかってそのまま床へ崩れ落ちた。
「お前……」
　父親は、理解できない化け物でも見るかのように這いずってアルトに近づき、その肩を震える手で掴んだ。「自分が……自分の姿が……ッ！ わかっているのか！！」

　父親の悲鳴に近い響きに、アルトはゆっくりと首を傾けた。
　ふと、手洗い場に備え付けられた小さな鏡が目に入る。
　そこに映っていたのは、特進クラスの制服を着た十六歳の少年の姿ではなかった。
　白髪交じりの痛んだ髪。深く窪んだ眼窩。肌は枯れ木のように乾燥し、首筋には無数の皺が寄っている。完全に精根尽き果てて老いぼれた四十代半ばの『疲れ切った中年の男』が、鏡の中からこちらを呆然と見つめ返していたのだ。

「あ……」
　声を聞いて、自分でも驚いた。これまでの高い少年の声ではなく、低く野太い、ひどく老いぼれた他人の声だった。

「ああ……ああぁっ……！」
　父親はアルトの制服の胸ぐらを両手で掴んだまま、ぐしゃぐしゃに顔を歪めて泣き叫んだ。
「ニーナが……娘が助かって……嬉しいに決まっている……！ 分かっている……お前が、ニーナの命を助けてくれたんだ……ッ！ だが……どうしてお前が、こんな姿にならなきゃならないんだ……ッ！！ 息子のお前が……自分の『未来』を全部引き換えにしてまで……ッ！！」
　愛する娘の命を取り戻した代償として、愛する息子の人生そのものを完全に喪失してしまった親の、逃げ場のない愛と絶望の絶叫だった。

「ごめん、父さん。でも僕は……ニーナを……家族を救えたんだから……これで、よかったんだよ……」
　親の悲しみを和らげようと必死に笑いかけて説明するアルト。しかし、その優しすぎる掠れた老声は、狂うように泣き崩れる両親の耳には、もはや残酷な呪いとしてしか届かなかった。

　激しく雨が窓ガラスを打ち据える冷たい診療室の中。
　外見が四十歳を過ぎた男になってしまった特進の少年は、永遠に泣き続ける愛する両親を見下ろしながら、ただ一人、致命的なまでの孤独の中に立ち尽くしていた。
