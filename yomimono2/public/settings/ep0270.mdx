# 第27話：演算

<!-- POV: アルト -->

　スラムの淀んだ空気を切り裂くように、狭い路地裏の奥で鈍い爆発音が響き渡った。

「っ……！ 逃げるなよ……こちとら手ぶらじゃ帰れねえんだよ……！」

　アルトは五人の男たちに包囲されていた。いずれも、ローガンの縄張りを狙っている新興の敵対組織の連中だ。狙いは一つ、ローガンの強大な権力の土台であり「どんな病気も治す」と噂されるアルトを拉致し、自分たちの組織に高く売り飛ばすことだろう。

「やめなよ。君たち、もう魔力毒が回っている。これ以上無理に魔法を使えば——」
「黙れ！ てめえを浚って高く売れば、俺たちも下っ端からオサラバできるんだよ！」

　アルトの形ばかりの忠告も虚しく、切羽詰まった五人の男たちは一斉に安物の魔導書を取り出し、無茶な火球の詠唱を開始した。
　四方八方から同時に放たれた熱波が迫る。
　アルトは瓦礫の山の上に張られた『完全透明化の結界』の中に、薄っすらとミアの気配を感じ取っていた。
　結界の奥で、空気が異常なほど冷え込んでいるのがわかる。彼女がこのチンピラどもを一瞬で氷漬けにしようと、致死の魔法を展開し始めている気配だった。
　（……だめだ。彼女にここで手出しさせるわけにはいかない）
　彼は決してそちらへは視線を向けず、無防備な立ち姿のまま静かに息を吐いた。
　（……仕方ない。これなら、ただの自滅に見えるはずだ）

　アルトは一切の防御陣を張らなかった。
　その代わり、懐から数枚の切れ端のようなメモ用紙を取り出し、無造作に空中に投げ捨てる。
　それと同時に、自身の魔力経路から流し出した微小な魔力が、用紙の上の魔法陣を一斉に空中に立体展開させた。
　詠唱の行程を完全に無視したそれらは、本来なら一瞬で暴走して自壊するはずの粗悪な陣だ。
　だが、アルトの指先は、空間に生じた魔法の『綻び』の感覚を正確に捉えていた。彼はその破綻して自壊するまでの間隙をわずかに削り取り、狂いなく極小の歯車のように噛み合わせてゆく。

　一つ目の陣が、敵の火球の熱を吸い上げて爆発する。
　（わずかに右——熱量〇・四、風圧の偏りを補正）
　こめかみに鋭い痛みが走る。五人から放たれた火球の全ての軌道、魔力密度、起爆時間を『ほんの二秒』の間に逆算し尽くし、寸分違わぬミリ単位の間合いで二つ目、三つ目の陣へと起動条件を連鎖させていく。

　連鎖的な爆発音と鋭い魔力の衝撃音が路地に響き渡る。

「なっ……！？ 魔法が、勝手に真っ直ぐ飛ばねえっ！？」

　配置された連鎖陣がビリヤードの球のように弾かれ、四人の放った火球が次々と同士討ちを引き起こす。だが、初めて実戦で試す極限の演算は決して完璧ではなかった。
　（遅れた……！）
　最後の陣の起動がコンマ数秒遅れ、五発目の火球の爆風が完全には逸れずにアルトの左腕を深々と舐め上げた。
　皮が焼け焦げ、肉が裂ける鈍い熱。
　だが、跳ね返った四発分の魔力の衝撃波がチンピラ全員の顔面を正確に打ち抜き、彼らは白目を剥いて次々と崩れ落ちた。

「ふぅ……みんな、大怪我じゃなくてよかった。今は少し眠っててね」

　気を失った連中を覗き込みながら、アルトはゆっくりと息を吐き出した。
　腹の底から湧き上がるどす黒い殺意と、左腕の肉が焼け焦げる凄まじい激痛を必死に顔の裏側へと押し殺し、無理やり顔の筋肉を引き上げる。
　そうして貼り付けられたのは、いつも通りの「無知で呑気なお人好し」の笑顔だった。
　残り一年半の死線を完璧な「ピエロ」として演じ切るためには、誰の目があるか分からない路地裏ですら、決して隙を見せるわけにはいかないのだ。

　アルトはローブの陰で、左腕の激痛を堪えるために血が滲むほど固く握りしめていた拳を、誰にも見えないようにゆっくりと解いた。
　透明な結界の中から、氷のようなミアの冷たい視線が自分へと突き刺さっているのを感じていた。だがアルトは、ただの「馬鹿な善人」の顔のまま、背を向けて路地裏の角を曲がった。

　だが、その張り付いた笑顔を保てたのも、ミアの視界から完全に外れるまでだった。
　物陰に入った瞬間、頭蓋骨の奥で限界を超えた脳と魔力経路がショートし、爆発的な耳鳴りが鳴り響いた。
「がっ……ぅは……っ」
　眼球の奥が焼け焦げるような激痛に視界が明滅し、強烈な吐き気と汚れのような睡魔に襲われ、アルトは堪えきれずに煤けた壁へと崩れ落ちる。

　だが、倒れている余裕はない。この焼け焦げた左腕のまま帰れば、間違いなくミアに勘付かれる。
　アルトは揺れる視界の中で右手を火傷の患部に押し当て、わずかに残された己の寿命を強引に削り落として、左腕の焼けただれた肉と皮膚を即席で塞ぎ切った。
　治癒の魔力を行使した反動が、限界を迎えていた身体に最後の一撃として深く突き刺さる。

　（帰らなきゃ……急いで、休まないと、脳が……）
　完全に意識が白濁していく中。ひどく震える足を引きずり、彼は死に物狂いで薄暗いあばら家への帰路についた。
