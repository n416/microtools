# 第22話：重なり合わない足跡

<!-- POV: アルト -->
　ローガンとの契約を受け入れ「スラムの裏医者」として活動を始めて数ヶ月が流れた。

　払う対価は以前のように使うたびに数十年が吹き飛ぶほど破滅的ではなくなった。だが絶え間なく吸い上げられ続ける『呪いの供出』は確実に彼の肉体を蝕み続けていた。
　王立院の歴史に名を刻むと信じられた十六歳の少年は、もうどこにもいない。今の彼の外見は深いシワが刻まれ、白髪が混じり丸まった背中で淀んだ空気を歩く「五十を手前にした初老の男」でしかなかった。

「……ふぅ……」

　スラム市場の淀んだ空気の中を、アルトは安い薬草の入った紙袋を抱え重い足取りで歩いていた。
　肉体が老いるとは呼吸が浅くなり関節が痛み、歩くことすら疲労の塊になることだ。
　それでも、アルト自身が助けたスラムの子供が元気に走り回る姿を見ると、少しだけ心が軽くなった。

　その時だった。

「すみません！ この少年を見かけませんでしたか！ 特進科の制服を着た、十六歳の……」

　薄暗い市場の入り口付近で、見慣れない清潔な衣服を着た初老の男性がすれ違う住人たちに頭を下げて回っていた。その手にはしわくちゃになった似顔絵のビラが握られている。
　アルトの足が影に縫い付けられたように硬直した。

「お兄ちゃんを、知りませんか……っ！ アルト兄さんを……！」

　男性の隣で泣き腫らした目でビラを配っているのはニーナだった。
　自らの寿命を代償にして死の淵から救い出した愛する妹は、間違いなく健康を取り戻し自らの足でしっかりと立っていた。

　それはアルトが命を削って望んだ結果のはずだ。だが——彼女はその取り戻した健康な足で、兄を探してこんなスラムを歩き回っている。退院して間もないはずの彼女の顔からかつての笑顔は消え去り、癒えない悲壮感に覆われているのだ。
　傍らで両手で顔を覆いしゃがみ込んで泣いているのは母親だった。
　かつて小鳥を作ってくれた父親の温かかった顔は過労と心労で震え土気色になっている。裕福で幸せだったはずの家族の姿は、息子を探すための心身の疲弊によって見る影もなくやつれ果てていた。

　アルトは声を出さなかった。
　ただ初老の己の姿と健康を取り戻した妹の姿を交互に見替えし、自身の選択がもたらしたあまりの残酷さに立ち尽くすことしかできなかった。
　『ここにいるよ』と軽々しく名乗り出ることなど、決して許されない。

　家族が配るビラの束が減っていくのを暗がりから息を殺して見つめていると、不意に、疲れ切った足取りの父親が傍らに立っていたアルトに気付いて頭を下げた。

「あの……すみません。そこのあなた。この少年を、スラムで見かけませんでしたか？ 私たちの……大事な、息子なんです」

　父親は目の前にいる初老の男が自身の愛する息子であることなど微塵も気付かぬまま、震える手で『十六歳の特進生アルト』の似顔絵を差し出した。
　アルトは息を呑んだ。
　間近で見る父親の目は充血し、母親は泣き声すら出せないほどに憔悴しきっていた。
　もしここで自分が「アルトだ」と名乗ったら何が起こるか。
　それは誰より自分自身が一番よくわかっていた。
　彼らは決して自身を許せないだろう。息子の莫大な命を犠牲にして生き延びたという罪悪感で発狂し、一生を地獄のような自責の念の中で過ごすことになる。

　彼らの記憶にある「アルト」は、輝かしい未来を持つ十六歳の天才少年のままでなければならない。いずれ数年で死にゆく「薄汚れの初老の不審者」の正体が自分だと知られるわけにはいかない。

（……ごめん。本当に、ごめん……！）

　アルトは唇を血がにじむほど強く噛み締めた。
　胸に渦巻く郷愁を、無理やり喉の奥へ呑み下す。
　そして顔を深くフードで隠し、何十年も煙草を吸い続けた老婆のようなしわがれた低い声を作った。

「……いや。知らないな。スラムで人探しなんてやめておけ。あんた達の来るような場所じゃない」
「……そうですか。急に引き留めて、申し訳ありませんでした」

　落胆の色を隠せずに深くお辞儀をした父親とともに、ニーナも「ありがとうございました」と小さく頭を下げた。
　立ち去ろうとした父親がふと足を止め、わずかにアルトの丸まった背中を振り返った。

「……いえ、失礼しました。一瞬、あの子の面影がしたような……かつての、澄んだ息子の声が聞こえたような気がして」

　その言葉に、アルトは心臓が抉られるような悲しみを感じた。
　たった二言三言の短い嘘の中にすら、長年聞き慣れた『無垢な十六歳のアルトの響き』が滲み出てしまったのだ。
　咄嗟にさらに猫背に背を曲げ、震えを誤魔化すように暗がりの中へ歩を進めた。

　父の温かい匂いも母の優しい声も、すがりつきたかった愛しい幻影は目の前を通り過ぎ無情にもスラムの霧深い闇へと消えていく。
　（二度とこんな淀んだ場所に足を踏み入れず、あの陽の当たる温かな家で、僕の悲惨な真実など一生知らずに笑って生きていけますように……）

　誰もいない暗い路地に逃げ込んだアルトは足の力が抜け、汚れた壁にズルズルと背を預けて崩れ落ちた。
　ひび割れてボロボロになった初老の顔を両手で深く覆う。声なき嗚咽だけがいつまでも彼の丸まった薄い肩を激しく震わせていた。
