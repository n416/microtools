# 0100『魔女の懺悔と、魂の還る場所』

　スラムの冷たい泥壁に囲まれた小部屋で、アルトは激しい熱を出して寝込んでいた。
　『年間最低五年分』という最低利用ノルマの生命力を初回で一気に吸い上げられた反動である。その肉体はたった一日で、白髪の混じる五十代手前の初老の姿へと、さらに残酷な老化を遂げていた。

「……だから忠告したんだ、この大馬鹿が」

　ミアはベッドの脇に座り込み、水瓶からすくった冷水を乱暴にアルトの口元に流し込んでいた。
　三百年間、誰の看病などしたこともない。こぼれた水がアルトの顔を濡らし、ひどく不機嫌そうに舌打ちを漏らす。
　だが、その氷のように冷たい声とは裏腹に、彼女はその場から離れようとはしなかった。

　熱に浮かされ、苦しげに息を吐く初老の男の顔。
　自分が搾取され、命を削られていると分かっていて、一切の怒りも持たずに他者のためにすり減っていく男。
　その理解不能な『過剰な利他性（狂気）』を見下ろしているうちに、ミアの胸の奥で、蓋をしていたはずの三百年前の古い記憶が嫌な音を立てて蘇ってきた。

（……本当に、どいつもこいつも馬鹿ばかりだ。あの時のあいつらと同じだ）

　ミアは薄暗い天井を仰ぎ、まるで独り言のように、静かに口を開いた。

「私は魔女だ。……他人の寿命（いのち）を代用（コスト）にさせ、自分の命はコンマ一秒すら使わずに、三百年ものあいだ生き延びてきた最悪の化け物だ」
「……」
「権力者や大富豪たちに『奇跡』を売り捌き、魔法の代償はすべて彼ら自身やその奴隷に払わせてきた。……私の計算式（効率）なら、彼らを救う魔法などたった数日の消費で済む。だが私は、見ず知らずの他人のために自分の命（タンク）をたった一秒でも削るのが怖かった。だから彼ら自身の命を容赦なく燃やさせて魔法を売り捌き、自分自身の莫大な寿命だけを無傷の『装甲』として温存してきた。……私は、あのヴィランよりも悪辣な、ただの冷酷な守銭奴（化け物）なんだよ」

「……それで？」

　アルトが目を閉じたまま、熱い呼気を吐き出してそう返した。軽蔑されると思っていたミアは言葉を詰まらせ、少しだけ悲しそうに顔を歪めた。

「だがな、三百年前のあの日……私を逃がすために盾になって死んだ同胞たちは、誰も私を恨まなかった。お前みたいに、笑って死んでいったんだ」

　ミアは自分の膝の上で両手を強く組み、誰にも言わなかった己の『真の目的』を吐き出した。

「私は、ただ死に損なっただけだ。だから……あの過去に戻って、あいつらと同じ場所できちんと死ぬ。それが、私が三百年他人の命を奪ってでも生き汚く生き延びてきた、ただ一つの目的だ。そのための『時空の逆行』……それを成し遂げるためだけに、膨大な理論と計算を積み上げてきた」

　ずっと解けなかった。空間と時間の因果を遡り、過去の座標を特定するための途方もない演算。
　自分の最高の知恵袋（センセイ）であるアルトに教えを乞えば、あるいはその数式が完成するのではないかと、ずっと期待していた。

「……計算なんて、要らないよ、ミア」

　不意に、掠れた声が響いた。
　驚いて見下ろすと、アルトが薄く目を開け、ひどく穏やかな瞳でミアを見つめ返していた。

「な、起きていたのか、貴様」
「空間や時間を超えるのに、難しい計算は要らない。……帰りたいと強く願う場所があるなら、君の『魂のアンカー』は、もう三百年前のその場所に真っ直ぐ刺さってるんだ」

　アルトは熱を持った手で、ベッドの端を握りしめた。
　エリート特待生としてのいかなる高度な魔法理論でもない。それは、ただ愛する妻の病を治したいと願い、全てを失って泥を這いずり回ってきた彼だけがたどり着いた、魔法の極めて原始的な本質だった。

「魔法は、祈りと代償の等価交換だ。……魂のアンカーが刺さっているなら、あとはそれに繋ぐだけの、途方もなく巨大な『魔力（いのち）』を一気に燃やせばいい。そうすれば、魂は必ずそこへ帰れる」

「途方もない……命」

　ミアは息を呑んだ。
　三百年間、数字と理論だけで魔法を解釈してきた彼女にとって、それはあまりにも非論理的で、だが決して否定できない『真理』の響きを持っていた。

（そうだ……理論が足りないのではない。ひとりの人間の魂を過去に押し流すほどの、規格外の命価（エネルギー）が足りないから、計算が弾かれていただけだ）

「……もし、そうだとすれば」
　ミアは、かつての大戦で使われたという禁忌の兵器の存在を、古い記憶の底から思い起こしていた。
「かつて、大勢の人間の命を並列で奪い、奇跡に転換する『広域魔力炉』という最悪の代物が存在した。……もしその炉の莫大な『熱』があれば、私の魂のアンカーは、過去へと繋がるかもしれないな……」

「そっか。……よかったな、ミア」

　アルトはそれだけを言うと、ふっと安堵したように目を閉じ、再び深い眠りへと落ちていった。
　荒かった呼吸は、いつの間にか穏やかなものに変わっている。

　ミアはベッドの横でしばらく座り込み、自分の手を見つめていた。
　彼が示した、計算を必要としない『魂の繋がり』という魔法の結論。
　それが、やがて迎える最悪の運命の果てで、アルト自身を救い、そして殺す引き金（アンカー）になることなど、この時の二人には知る由もなかった。
