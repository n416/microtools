# 第40話：灰の少年と、ただの少女

　家までの長い帰り道。

「……ふふっ、なんだこれ。馬鹿みたいに体が軽いじゃないか」

　ミアは薄汚れた手を愛おしそうに見つめ、自嘲するようにけれど心の底から晴れやかに笑った。
　もう誰の命も奪わなくていい。何百手先を計算して効率に怯えながら防御を固めなくてもいい。これからは自分自身も残された有限な時間をただ不器用に使い果たしていくことができるのだから。
　かつての計算狂の魔女は今ここに完全に死んだのだ。

「本当に……無茶苦茶な魔法だったな」

　瓦礫の向こう側から足音が響いた。
　ミアが顔を上げると、朝日が差し込む崩れた天井の下に青年が立っていた。

　死を待つだけの老人の姿はどこにもない。
　そこには失われていた時間を正しく取り戻したばかりの、澄んだ瞳を持つ頼もしい青年の姿があった。
　ただ一つ、彼が背負った過酷な因果を証明する美しい灰色の髪を揺らして。

「気分はどうだ、アルト。……『転換炉』の適合具合は」
「完璧だよ。君が作ってくれた世界で一番贅沢な心臓のおかげでね」

　十八歳の逞しい青年へと育ったアルトが、奇跡を確かめるように胸に手を当てミアへ微笑んだ。
　その澄んだ強さを持った瞳がミアに確信させた。
　自分などどうでもいいと嗤う都合の良い『お人好し』の気配が、今の彼からは綺麗に消え去っている。過去の記憶と書き換えられた肯定の因果が、あの還元炉の光によって『どちらも今の自分を生かしている人間らしさ』へと完璧に統合されたのだと。

「……歩けるかい、ミア」
　アルトがバランスを取っている少女に向けて大きくなった手を差し伸べた。
「あぁ」
　ミアは計算など一切せず、その温かい手を迷わずに握りよろめきながらも立ち上がった。

「私たちは今日でこのスラムから出ていく。……帰りたいと言ったその『家』まで私が付いて行ってやる」
「ありがとう。……行こうか」

　二人は瓦礫の山を抜け、地表の光が差し込む長い上り坂へと並んで歩き出した。
　身を寄せ合うわけでもなくけれど互いの存在を確かに感じられる等間隔で。
　崩落した残骸を黙々と登っていく道中。
　ふとアルトが立ち止まった。

　先ほどまでの晴れやかな顔から一転して、彼は自分のために生力を使い切って無力になった小さな少女へ胸に刺さる核心を静かに問うた。

「なあ。……本当によかったのか」
「……」
「俺に生命力を全部使っちまって。お前は本当は、その命を使ってあの時代に戻りたかったんじゃないのか？」

　それはかつてミアが『私には過去に帰る未練がある。だから寿命を溜め込んでいる』と吐き捨てた彼女自身の【呪い】の言葉だった。
　アルトは自分のために彼女がその念願を手放してしまったことを重く受け止めて案じているのだ。

「俺に全部使っちまったら。……お前はもうあの時代には戻れないんだぞ」

　少しだけ震えるアルトの声。
　だがミアは足を止めず振り返りもしなかった。
　ただ瓦礫の向こう側から差し込む眩しい朝の光を真っ直ぐに見据えたまま、彼女の声はかつてないほどに柔らかく透き通っていた。

「……いい。構わない」

　永遠にも等しい孤独の中で、過去に向かって生き汚く計算を稼動し続けていた氷の魔女が。
　初めて明確な自分の意思で『今の時代』を選んだ、完全な肯定。

「私の未練よりも重い鎖が、もう……今、ここにあるからな」

　その言葉がひんやりとした明け方の空気に静かに溶けていく。
　後ろを歩いていたアルトは一瞬だけ足を止め――やがて安堵したように小さく笑い静かに頷いた。

「……そっか」

　そして二人は再びただの対等な不器用な人間として並んで歩き始めた。
　無償の善意を搾り取られていた灰色の少年と、世界の無駄を絶版しようとしていた防壁の少女。
　彼らはもう互いに何の装甲も計算もなく。
　降り注ぐ朝の光に満ちた広くどこまでも計算不可能な世界へと向かってただ一歩ずつ歩んでいった。
